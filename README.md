## üöÄ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Pkdriller/MY-PORTFOLIO-WEBSITE-.git
   cd MY-PORTFOLIO-WEBSITE-
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally**
   ```bash
   npm run dev
   ```

4. **Deploy on Vercel**
   - Click the button above to deploy directly to Vercel.
